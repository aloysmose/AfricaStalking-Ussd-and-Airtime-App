USSD APP THAT USES AFRICA'S TALKING API
------------------------------------------
-> Simple ussd app that consumes ussd and airtime api from africa's talking
-> App is used to perform a simple survey via ussd then it sends airtime to the first 10 respondents.
-> App is integrated with salesfroce crm to be able to fetch the user details from salesforce and display to the user.
-> App uses mysql db to be able to store user session and responses
